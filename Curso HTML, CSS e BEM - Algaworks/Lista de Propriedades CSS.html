A tabela contém um grande número de propriedades CSS, por isso caso deseje buscar alguma coisa, tecle CTRL+F e faça sua busca.

Propriedade CSS Função Possíveis valores Exemplo
background Define os parâmetros referentes ao fundo de um elemento. Aceita de 1 a 5 valores. Aceita valores de: background-color; background-image; background-repeat; background-attachment; background-position; background:url(minha_imagem.png) no-repeat center center;
background-attachment Define se a imagem aplicada como fundo de um elemento ficará fixa ou se irá rolar junto com o conteúdo da página. scroll; fixed; background-attachment:scroll;
background-color Define uma cor de fundo para um elemento. nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); transparent background-color:#0066ff;
background-image Define uma imagem como fundo de um elemento. url; none; background-image: url(minha_imagem.png);
background-position Define a posição da imagem aplicada como background. top left; top center; top right; center left; center center; center right; bottom left; bottom center bottom right; x% y%; unidades_css_x unidades_css_y background-position:center left;
background-repeat Define se a imagem vai repetir e a direção da repetição (X ou Y). repeat; no-repeat; repeat-x; repeat-y; background-repeat:repeat-x;
border Define estilos para as 4 bordas de um elemento. Aceita de 1 a 3 valores. Aceita valores de: border-style; border-width; border-color; border:#000 solid 2px;
border-collapse Define se as bordas das células da tabela serão fundidos ou mantidos separados. collapse; separate; border-collapse:separate;
border-color Define a cor das 4 bordas de um elemento. Aceita de 1 a 4 valores. nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); border-color:blue;
border-radius Define o arredondamento dos quatro cantos de um elemento. Aceita 4 valores, o primeiro para o canto superior esquerdo, e os outros valores para os demais cantos seguindo o sentido horário. unidades_css border-radius: 0px 20px 0px 20px;
border-spacing Define a distância entre as bordas de células adjacentes. Só aplicável quando border-collapse for separate. unidades_css border-spacing:5px;
border-style Define o estilo das 4 bordas de um elemento. Aceita de 1 a 4 valores. none; hidden; dotted; dashed; solid; double; groove; ridge; inset; outset; border-style:dashed;
border-top; border-right; border-bottom; border-left; Define estilos para as bordas superior, direita, inferior e esquerda. Aceitam de 1 a 3 valores. Aceita valores de: border-style; border-width; border-color; border-left:#333 solid 4px;
border-width Define a espessura das 4 bordas dos elementos. Aceita de 1 a 4 valores. thin; medium; thick; unidades_css; border-width:8px;
box-shadow Define uma sombra a um elemento. Aceita 4 valores, sendo um para a cor e os demais para as medidas da sombra. A primeira unidade CSS representa o deslocamento horizontal da sombra, o segundo o deslocamento vertical, e o terceiro a dissipação da sombra. unidades_css; nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); box-shadow: -5px 5px 3px #000;
caption-side Define a posição da legenda na tabela. top; bottom; left; right; caption-side:top;
clear Define em quais os lados de um elemento que não são permitidos elementos flutuantes. left; right; both; none; clear:both;
clip Define a forma de um elemento. auto; shape; clip:shape;
color Define a cor do texto. nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); color:#999;
column-count Define o número de colunas para o elemento. número; column-count:2;
column-gap Define o espaço entre as colunas (padding). unidades_css; column-gap:5px;
column-rule Define a estilização da borda que ficará entre as colunas. Aceita valores de: border-style; border-width; border-color; column-rule: 2px solid #06f;
content Gera conteúdo para um elemento. Utilizada com os pseudo-seletores :before e :after. string; url; counter(name); counter(name, list-style-type); counters(name, string); counters(name, string, list-style-type); attr(X); open-quote; close-quote; no-open-quote; no-close-quote; content:no-close-quote;
counter-increment Define em quanto será incrementado o contador a cada ocorrência de um seletor. none; número identificador; counter-increment:none;
counter-reset Define o valor a ser atribuído ao contador a cada ocorrência de um seletor. none; número identificador; counter-reset:none;
cursor Define um formato ou imagem para o cursor do mouse. url; auto; crosshair; default; pointer; move; e-resize; ne-resize; nw-resize; n-resize; se-resize; sw-resize; s-resize; w-resize; text; wait; help; cursor:url(meu_ponteiro.gif);
direction Define a direção em que é escrito o texto. Da esquerda para a direita ou da direita para a esquerda. ltr; rtl; direction:rtl;
display Define se um elemento deve ser apresentado ou não, e a sua forma de apresentação. none; inline; block; list-item; run-in; compact; marker; table; inline-table; table-row-group; table-header-group; table-footer-group; table-row; table-column-group; table-column; table-cell; table-caption; display:inline;
empty-cells Define se as células sem conteúdo visível devem ter bordas ou não. Só aplicável quando border-collapse for separate. show; hide; empty-cells:show;
float Define o alinhamento de um elemento e controla o fluxo de texto ou de outros elementos ao seu redor, além de tornar um elemento flutuante. left; right; none; float:right;
font Define os estilos da fonte. caption; icon; menu; message-box; small-caption; status-bar; e também aceita valores de: font-family; font-size; font-style; font-weight; font-variant; font:Arial #06f bold;
font-family Define o tipo de fonte. nome_da_familia; nome_da_fonte; font-family:Arial;
font-size Define o tamanho da fonte. xx-small; x-small; small; medium; large; x-large; xx-large; smaller; larger; unidades_css; %; font-size:12px;
font-size-adjust Define que o tamanho da fonte deve ser ajustado, caso o browser tenha de utilizar outro tipo de fonte que não foi o escolhido em font-family. none; número; font-size-adjust:none;
font-stretch Define uma contração ou expansão na largura da fonte. normal; wider; narrower; ultra-condensed; extra-condensed; condensed; semi-condensed; semi-expanded; expanded; extra-expanded; ultra-expanded; font-stretch:normal;
font-style Define o estilo da fonte. normal; italic; oblique; font-style:italic;
font-variant Define se deve ser utilizado o tipo de fonte small-caps ou normal. normal; small-caps; font-variant:small-caps;
font-weight Define a espessura do traço da fonte. normal; bold; bolder; lighter; 100; 200; 300; 400; 500; 600; 700; 800; 900; font-weight:bold;
letter-spacing; word-spacing; Definem o espaço entre os caracteres ou entre as palavras de um texto. normal; unidades_css; letter-spacing:1px;
line-height Define a altura das linhas de texto. normal; número; unidades_css; %; line-height:25px;
list-style Define todos os parâmetros referentes a uma lista. Aceita de 1 a 3 valores. Aceita valores de: list-style-type; list-style-position; list-style-image; list-style:none;
list-style-image Define uma imagem a ser utilizada como marcador de item de uma lista. none; url; list-style-image:url(marcador.png);
list-style-position Define a posição do marcador de item da lista. inside; outside; list-style-position:inside;
list-style-type Define o tipo do marcador de item da lista. none; disc; circle; square; decimal; decimal-leading-zero; lower-roman; upper-roman; lower-alpha; upper-alpha; lower-greek; lower-latin; upper-latin; hebrew; armenian; georgian; cjk-ideographic; hiragana; katakana; hiragana-iroha; katakana-iroha; list-style-type:circle;
margin Define as 4 margens do elemento. Aceita de 1 a 4 valores. Aceita valores de: margin-top; margin-right; margin-bottom; margin-left; margin:15px 5px 5px 15px;
margin-top; margin-right; margin-bottom; margin-left; Definem as margens superior, direita, inferior e esquerda de um elemento. auto; unidades_css; %; margin-top:50%;
max-width; max-height; Definem a largura máxima e a altura máxima de um elemento. none; unidades_css; %; max-height:200px;
min-width; min-height; Definem a largura mínima e a altura mínima de um elemento. unidades_css; %; min-width:100px;
outline Define o estilo para as linhas de contorno. Aceita de 1 a 3 valores. Aceita valores de: outline-style; outline-width; outline-color; outline:3px solid #000;
outline-color Define a cor das linhas de contorno. invert; nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); outline-color:#262626;
outline-offset Define a distância das linhas de contorno em relação as bordas do elemento. unidades_css; outline-offset: 10px;
outline-style Define o estilo das linhas de contorno. none; hidden; dotted; dashed; solid; double; groove; ridge; inset; outset; outline-style:dashed;
outline-width Define a espessura das linhas de contorno. thin; medium; thick; unidades_css; outline-width:2px;
overflow Define o que deve ocorrer quando o conteúdo de um elemento excede sua área. visible; hidden; scroll; auto; overflow:hidden;
padding Define o espaço entre o conteúdo e os 4 limites ou bordas de um elemento. Aceita de 1 a 4 valores. Aceita valores de: padding-top; padding-right; padding-bottom; padding-left; padding:5px 3px 5px 30px;
padding-top; padding-right; padding-bottom; padding-left; Definem o espaço entre o conteúdo e os limites ou bordas superior, direito, inferior e esquerdo. unidades_css; %; padding-right:10px;
position Define a posição de um elemento. static; relative; absolute; fixed; position:fixed;
quotes Define o estilo das aspas. none; string string; quotes:none;
table-layout Define como será desenhada a tabela. auto; fixed; table-layout:fixed;
text-align Define o alinhamento do texto. left; right; center; justify; text-align:center;
text-decoration Define um efeito de decoração para o texto. none; underline; overline; line-through; blink; text-decoration:none;
text-indent Define um deslocamento para a direita ou esquerda da primeira linha do texto. unidades_css; %; text-indent:10px;
text-shadow Define uma sombra para o texto contido no elemento. Aceita 4 valores, sendo um para a cor e os demais para as medidas da sombra. A primeira unidade CSS representa o deslocamento horizontal da sombra, o segundo o deslocamento vertical, e o terceiro a dissipação da sombra. unidades_css; nome_da_cor; #rrggbb; #rgb; rgb(d,d,d); rgb(d%,d%,d%); text-shadow: 1px 1px 0px #000;
text-transform Define que o texto deve ser capitalizado ou ser exibido todo em maiúsculas ou minúsculas. none; capitalize; uppercase; lowercase; text-transform:capitalize;
top; right; bottom; left; Definem a distância do elemento em relação ao limite do elemento que o contém. auto; unidades_css; %; top:10px;
transition-duration Define o tempo do efeito de transição aplicado a propriedade CSS em transition-property. unidades_de_tempo; (segundos) transition-duration: 0.5s;
transition-property Define a qual propriedade CSS deve ser aplicado o efeito de transição. nome_de_propriedades_css; transition-property: color;
vertical-align Define o alinhamento vertical do elemento. baseline; sub; super; top; text-top; middle; bottom; text-bottom; unidades_css; %; vertical-align:sub;
visibility Define se um elemento deve ficar visível ou não. visible; hidden; collapse; visibility:visible;
white-space Define o tratamento a ser dado ao espaço em branco existente dentro de um elemento. normal; pre; nowrap; white-space:pre;
width; height; Definem a largura e a altura de um elemento. auto; unidades_css; %; width:100%;
z-index Define a ordem de apresentação dos elementos quando há sobreposição. auto; número; z-index:5;
