Esta DIV utiliza a propriedade { padding : 50px;} para gerar espaços entre o conteúdo e a div. Podemos variar esta propridade com padding-top, padding-right, padding-left, padding-bottom
Esta DIV utiliza a propriedade { margin : 50px;} para adicionar uma margem entre a div filha e a div pai Podemos variar esta propridade com margin-top, margin-right, margin-left, margin-bottom
As medidas pode ser expressas em %, por exemplo: margin : 50%
